# Project Overview

## Goal

Build a tool that any coding agent can use to autonomously identify bottle necks & gaps in tooling required to create an autonomous software factory. 

The primary goal of this agent is to educate teams on what is needed, and provide a plan to get easy wins.

Anyone should be able to run this, can get immediate value.

---

## Project Name Ideas

- **Mentat:**  From Dune universe. A "human computer" that processes massive data to find the logical "Golden Path." It fits the idea of an instructor providing insights humans can’t see.
- **Hephaestus:** The god who created the first **automata** (self-operating machines). It represents the transition from manual labor to a "Software Factory."
- **Talos:** In myth, a giant automated bronze man. It represents the end-goal: a system that protects and maintains itself.
- **Event Horizon:** The boundary where the laws of the system change. It represents the "Point of No Return" where a company moves from manual work to total automation.

## Form factor

*Unknown at the moment. But here are some candidates*

- Skills: Can run on any coding agent - Claude Code, Codex, OpenCode, etc. Also can be published to [skills.sh](http://skills.sh) for discovery
- Plugin: Allows us to package multiple skills, subagents, MCPs, etc. A more complete solution, but only works with Claude Code. Allow us to create multiple slash commands, one for each skill.
- CLI: We will create a CLI regardless because it will make it easy for us to create tools for the agent that can do some complex work under the hood.
- Website: Something where users can publish and see the score. This is a must for virality.
- Blog / Writeup: A nice article that can be an entry point for companies to understand our philosophy (something like this with nice animations - [https://background-agents.com/](https://background-agents.com/))
