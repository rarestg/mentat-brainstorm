# The AI software engineer you can rely on

Ona works with and for your teams across the entire development lifecycle. Automatically set up with your code, secrets, and policies.

Run in [our cloud](/ona/runners/ona-cloud) or [your VPC](/ona/runners/aws/overview) with full control over security and compliance.

## What is Ona?

* **Environments**: Secure, ephemeral, pre-configured development environments that spin up instantly.
* **Agents**: AI software engineers that execute tasks in parallel or autonomously with full codebase context.
* **Guardrails**: Security, compliance, and governance controls to deploy at enterprise scale.

## Quick start

**1) Set up environments and runners**

* Choose where to run: [Ona Cloud](/ona/runners/ona-cloud) or your VPC ([AWS](/ona/runners/aws/overview), [GCP](/ona/runners/gcp/overview))
* Configure [Dev Containers](/ona/configuration/devcontainer/getting-started) and [Tasks and Services](/ona/configuration/tasks-and-services/overview)

**2) Configure integrations**

* Source control: [GitHub](/ona/source-control/github), [GitLab](/ona/source-control/gitlab), [Bitbucket](/ona/source-control/bitbucket), [Azure DevOps](/ona/source-control/azuredevops)
* IDEs: [VS Code](/ona/editors/vscode), [Cursor](/ona/editors/cursor), [JetBrains](/ona/editors/jetbrains), [Zed](/ona/editors/zed)
* Tools: [Linear](/ona/agents/integrations#linear), [CLI](/ona/integrations/cli), [Browser extension](/ona/integrations/browser-extension)

**3) Get productive with Ona Agent**

* Learn capabilities and best practices: [Ona Agent](/ona/agents/overview)
* Add team conventions with [AGENTS.md](/ona/agents-md)
* Create [Slash commands](/ona/slash-commands) and use [Bash commands](/ona/bash-commands)

**4) Use guardrails**

* Enforce policies and governance: [Guardrails](/ona/guardrails/overview)
* Audit changes: [Audit logs](/ona/audit-logs/overview)
* Control agent execution: [Command deny list](/ona/command-deny-list)
* Enterprise identity: [SSO overview](/ona/sso/overview), [OIDC](/ona/configuration/oidc)
